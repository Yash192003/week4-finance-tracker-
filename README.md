# week4-finance-tracker-
Personal Finance Tracker is a comprehensive command-line application that helps users manage their expenses, categorize spending, and generate insightful reports. It uses Python file handling (JSON and CSV), modular code structure, and robust error handling to provide a practical, real-world finance tracking tool.

## Features

- Add, view, and search expenses
- Categorize spending (Food, Transport, Bills, etc.)
- Generate monthly reports
- Set and track monthly budgets
- Export data to CSV
- Backup and restore data
- View statistics

## How to Run

```bash
cd week4-finance-tracker
python run.py
